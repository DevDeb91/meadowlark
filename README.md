# Meadowlark Travel

Sample website built while studying **[Web Development with Node & Express](https://www.oreilly.com/library/view/web-development-with/9781492053507/)** (2nd edition) by Ethan Brown — used as a running example throughout the book.

## Tech stack

- Node.js
- Express
- Handlebars (view engine)

## Getting started

```bash
npm install
node meadowlark.js
```

Then open http://localhost:3000
